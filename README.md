Portfolio for Mayibongwe Khumalo

To host live (recommended: GitHub Pages):

1. Create a Git repository and commit the `portfolio_mayibongwe` folder.
2. Push to GitHub and enable GitHub Pages from the repository Settings → Pages → Source → `main` branch / root.
3. After a minute the site will be available at `https://<your-username>.github.io/<repo-name>/portfolio_mayibongwe/` or if you push the site to the repo root it will be at `https://<your-username>.github.io/`.

Alternative quick hosts:
- Netlify: drag-and-drop the folder in the Netlify Deploy page.
- Vercel: create a new project and import the repo.

Edit `index.html` to add your real email, GitHub and LinkedIn links, and project demos.
